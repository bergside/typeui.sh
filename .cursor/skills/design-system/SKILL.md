# flowbite test Design System Skill (Cursor)

## Brand
flowbite is a ui component on top of tailwind css, it is a project used by over 30 million projects

## Style Foundations
- Visual style: modern, clean, high-contrast
- Typography scale: 12/14/16/20/24/32
- Color palette: primary, neutral, success, warning, danger
- Spacing scale: 4/8/12/16/24/32

## Component Families
- buttons
- inputs
- cards
- modals

## Accessibility
WCAG 2.2 AA, keyboard-first interactions, clear focus states

## Writing Tone
concise, confident, and helpful

## Rules: Do
- prefer consistent spacing
- use semantic colors
- preserve hierarchy

## Rules: Don't
- avoid low contrast
- avoid inconsistent radii
- avoid vague labels

## Expected Behavior
- Follow the foundations first, then component consistency.
- When uncertain, prioritize accessibility and clarity over novelty.
